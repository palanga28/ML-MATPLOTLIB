ce projet est a deposer apres le retour a la faculté. 
cette fois-ci on ne laisse rien passer les amis. 

voici quelques instructions qui vons nous permettre de bien travailler avec cet outil (GitHub):

à faire pour créer sa branche comportant son nom, selon qu'il nous a été demandé.

1.git checkout main
2.git pull
3.git checkout -b prenom
4.git push origin prenom

faites les ameliorations et modifications uniquement sur votre branche.
Vérifiez sur quelle branche vous êtes :

1.git branch
après s'être rassuré il faut maintenant Ajouter vos modifications :
2.git add .
apres avoir fait le git add . il faut toujours faire Commit vos changements :

git commit -m "Description de la modification ou amélioration apporté. ceci vas 
me permettre de comprendre en bref ce que vous avez fait et permettre aussi au 
professseur de comprendre ce que vous avez apporter au travail du groupe".

Envoyer vos modifications sur GitHub :

3.git push origin prenom 
NB: je vous en prie Toujours vérifier que vous êtes sur votre branche, pas sur main.
